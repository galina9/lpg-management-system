# LPG Management System

LPG Management System is a web application for managing a liquefied petroleum gas (LPG) business.

## Features

- User Authentication
- Dashboard
- Product Management
- Customer Management
- Orders Management
- Employee Management
- Multi-language Support
  - English
  - Armenian
  - Russian
- Responsive Admin Panel
- Bootstrap 5 UI
- Laravel 13
- MySQL Database

## Technologies

- PHP 8.3
- Laravel 13
- Bootstrap 5
- MySQL
- JavaScript
- Vite

## Installation

```bash
git clone git@github.com:galina9/lpg-management-system.git

cd lpg-management-system

composer install

npm install

cp .env.example .env

php artisan key:generate

php artisan migrate

npm run dev

php artisan serve
```
